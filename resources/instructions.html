Instructions

Objective

Identify the difference between the two images in the selected set by clicking and releasing the mouse on the location of the change. Differences could be:

  • Presence or absence of items.
  • Change in color.
  • Change in location.
  • Change in size.
  • Etc..

Start or Stop a Trial by clicking and releasing one of the images along the bottom of the application window.

If a log file has not been created using the "Create Log File" option, in the same menu as these instructions, the user must respond to a pop-up message box asking "Proceed without Trail Log?". There are three options:

  • Yes - the trial will proceed with selected parameters without a log.
  • No - A file dialog window will open to allow the user to specify the file name and directory for a trial log. Once the log is created, the user may launch a trial by clicking on the image again.
  • Cancel - This cancel the trial. The user may then consult the instructions again or exit the program.

The Log is a CSV (Comma Separated Variables) file. It appends trial records to the file. Therefore the same CSV may be used to record different trials at different times etc..

Mask Time is the length of time a gray mask is displayed between the two images in a set. The value may be changed by selecting the desired duration form the drop down, then pressing and releasing the "Set" button.

Alteration Speed is how quickly the two images in a set alternate. The value may be changed by selecting the desired speed form the drop down, then pressing and releasing the "Set" button.

Trial Timer displays the elapsed time for the currently running trail or the total elapsed time for the last trial run.

Incorrect Clicks displays the total number on incorrect attempts to identify the difference for the currently running trail or for the last trial run.
